dashboard

musical_analysis_app.pages.dashboard.main()[source]

The main function that sets up and runs the musical analysis dashboard.

This function initializes the Streamlit app with a set of interactive tabs that allow users to explore different aspects of musical track data, including popularity, musical characteristics, trends, and recommendations to maximize sales.

The function does the following: - Configures the page layout and title. - Loads the dataset and normalizes the tempo feature. - Displays a genre selection widget for users to highlight a specific genre. - Organizes and displays various analysis tabs:

  • “Popularité” (Popularity): Analyzes average popularity by genre.

  • “Caractéristiques” (Characteristics): Analyzes musical features by genre.

  • “Tendances” (Trends): Analyzes average track duration by genre.

  • “Recommandations” (Recommendations): Provides recommendations for maximizing track sales based on characteristics of popular tracks.

Each tab presents its analysis using Streamlit components and Plotly charts.

The function runs within a Streamlit app and interacts with the data using pandas, while also making use of Plotly for creating interactive visualizations.

Raises:

ValueError – If the genre selection is invalid or data is missing.

Returns:

None

Return type:

None

caracteristics_tab

musical_analysis_app.dashboard_tabs.caracteristics_tab.caracteristics_tab(df)[source]

Display the ‘Caractéristiques Musicales’ tab with visualizations.

This function generates two main plots: a bar chart and a radar chart.

The bar chart shows the average selected musical feature for each genre. The radar chart shows the profile of a selected genre based on the average values of various features (excluding “tempo”).

Parameters:

df (pandas.DataFrame) – The dataframe containing the music track data, including the columns ‘track_genre’ and various musical features.

The function utilizes Plotly for visualizations and Streamlit to render the UI.

recommendations_tab

musical_analysis_app.dashboard_tabs.recommendations_tab.recommendations_tab(df)[source]

Display the ‘Recommandations pour Maximiser les Ventes’ tab with insights on the optimal characteristics of successful tracks to help maximize sales.

This function calculates and displays the optimal features for music tracks based on their popularity, both globally and by genre. It provides actionable insights on how to create tracks that align with successful trends in the music industry.

Parameters: df (pandas.DataFrame): The dataframe containing the music track data, including

the columns ‘popularity’, ‘track_genre’, and the feature columns.

The function uses Streamlit to display the metrics for successful track characteristics and the features specific to the selected genre.

tendances_tab

musical_analysis_app.dashboard_tabs.tendances_tab.tendances_tab(df)[source]

Display the ‘Analyse des Tendances’ tab with insights on the average duration of tracks by genre, helping identify trends in track length across genres.

This function calculates and visualizes the average track duration for each genre and highlights the selected genre for comparison. The analysis helps to uncover trends in the duration of tracks that might be associated with genre-specific popularity or success.

Parameters: df (pandas.DataFrame): The dataframe containing the music track data, including

the ‘track_genre’ and ‘duration_ms’ columns.

The function uses Streamlit to display a bar chart showing the average track duration by genre, with a specific color for the genre selected by the user.
